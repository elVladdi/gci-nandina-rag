## Criterios de inclusión

- Documentos públicos o autorizados.
- Documentos vinculados con NANDINA, Sistema Armonizado, reglas generales, notas legales o referencias normativas relevantes.
- Documentos con procedencia identificable y versión registrable.

## Criterios de exclusión

- Documentos sin fuente verificable.
- Documentos duplicados o inconsistentes.
- Archivos corruptos o no convertibles a texto útil.
- Documentos no autorizados para uso académico o experimental.
